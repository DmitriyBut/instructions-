# Подсказки по командной строке

Команда смены директории
```sh
cd c:\folder_name
```

Команда отбражения текущей директории. MacOs, Linux
```sh
pwd
```

листинг текущей директории
Windows:
```sh
dir
```
linux, MacOs:
```sh
ls
```

Удаление файла Windows:
```sh
del <filename>
```
Linux, MacOs:
```sh
rm <filename>
```
